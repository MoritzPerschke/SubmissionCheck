```
Usage: check_submissions [options]
    -d, --dir DIRECTORY              Specify a directory to work in (default: current dir)
    -o, --output-dir OUTPUT_DIR      Specify a directory to output unzipped files to (default: './submissions')
    -c, --cs-identifier              Create directories named after students csXXXXXX identifier instead of their name
    -u, --remove-unwanted            Remove unwanted files (e.g. .vscode) in addition to warning about them. Unwanted extensions (.docx) will be kept
```

Written in a way to be added to path and executed in directory of zip file
